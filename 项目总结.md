## 🚀 快速开始

### 1. 启动应用
```bash
cd learn-java
mvn spring-boot:run
```

### 2. 访问系统
- **主页面**: http://localhost:8080
- **API文档**: http://localhost:8080/swagger-ui/index.html
- **H2控制台**: http://localhost:8080/h2-console

### 3. 使用示例
1. 打开 http://localhost:8080
2. 点击"生成数据"按钮生成模拟数据
3. 点击"刷新统计"查看统计信息
4. 使用其他按钮测试API功能
